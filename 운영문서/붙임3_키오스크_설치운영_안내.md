# 「군포시 민원 서식 작성 도우미」 키오스크 설치·운영 안내

〔붙임 3〕 · 작성일 2026. 7. 23. · 담당 〔부서명 / 담당자 / 연락처〕

> 본 도우미는 **인터넷 없이(오프라인) 키오스크 PC 자체에서 완전히 동작**합니다.
> 외부 서버·설치 프로그램·데이터베이스가 필요 없으며, 폴더 하나만 복사하면 됩니다.

---

## Ⅰ. 구성 파일 (배포 세트)

`키오스크배포` 폴더 안에 아래 파일만 있으면 됩니다. (총 8개, 약 2.4MB)

| 파일 | 설명 |
|------|------|
| `index.html` | 첫 화면(메뉴 허브) — 여기서 서식 선택 |
| `passport-helper-v1.html` | 여권발급신청서 |
| `marriage-helper-v1.html` | 혼인신고서 |
| `birth-helper-v1.html` | 출생신고서 |
| `death-helper-v1.html` | 사망신고서 |
| `naming-helper-v1.html` | 개명신고서 |
| `divorce-helper-v1.html` | 이혼(친권자 지정)신고서 |
| `민원도우미_실행.bat` | 키오스크 전체화면으로 여는 실행 파일 |

> 각 HTML 파일은 **그 자체로 완결**되어 있습니다(서식 이미지·글꼴·기능이 파일 안에 모두 포함). 인터넷·추가 설치 불필요.

---

## Ⅱ. 설치 방법

1. **키오스크 PC에 크롬(Google Chrome) 설치** (없을 경우 1회만).
2. `키오스크배포` 폴더를 키오스크 PC의 원하는 위치에 복사.
   - 예: `C:\민원도우미\`
3. 끝. (별도 설치 과정 없음)

---

## Ⅲ. 실행 방법

### 방법 A — 실행 파일(권장)
- `민원도우미_실행.bat` 를 더블클릭하면 **크롬이 전체화면(키오스크 모드)** 으로 첫 화면을 엽니다.
  - 자동으로 **시크릿(incognito) 모드**로 열려 방문 기록·다운로드 흔적이 남지 않습니다(개인정보 보호).
  - 주소창·뒤로가기가 감춰져 시민이 다른 사이트로 나가지 않습니다.
- **관리자가 종료할 때** : `Alt + F4` (전체화면 종료).

### 방법 B — 수동 실행
- 크롬에서 `index.html` 파일을 직접 열어도 됩니다(전체화면 잠금은 없음).

### 자동 시작(선택) — PC 켜면 자동 실행
1. `민원도우미_실행.bat` 우클릭 → [바로가기 만들기].
2. 만들어진 바로가기를 아래 폴더에 넣기:
   - `Win + R` → `shell:startup` 입력 → 열린 폴더에 바로가기 붙여넣기.
3. 다음 부팅부터 자동으로 도우미 화면이 뜹니다.

---

## Ⅳ. 프린터 설정 〔중요〕

인쇄물의 칸 위치·크기가 정확해야 여권 등 **기계 판독 오류가 나지 않습니다.**

- 키오스크에 연결된 프린터를 **기본 프린터**로 지정.
- 인쇄 시 아래 설정 확인(인쇄 직전 안내 팝업도 표시됨):
  - **용지 여백 : 없음**
  - **배율(크기) : 100% (실제 크기)** — ‘페이지에 맞춤’ 사용 금지
- 가능하면 크롬 인쇄 기본값을 위 설정으로 미리 맞춰 두면 시민이 매번 조정하지 않아도 됩니다.

## Ⅴ. 개인정보·보안

- 시민이 입력한 정보는 **PC 메모리에서만** 처리되고, 화면을 닫거나 새로고침하면 **즉시 사라집니다.** (저장·전송 없음)
- 실행 파일은 **시크릿 모드**로 열려 방문 기록·자동완성·다운로드 목록이 남지 않습니다.
- **인쇄가 원칙**이며, 부득이 PDF로 저장했다면 이용 후 파일을 삭제하도록 안내합니다.
- 외부 서버와 통신하지 않으므로 내부망·폐쇄망 환경에서도 사용 가능합니다.

- 자동 시작 설정·프린터 설정은 그대로 유지됩니다.

## Ⅶ. 문제 해결(FAQ)

| 증상 | 조치 |
|------|------|
| 실행 파일을 눌러도 안 열림 | 크롬 설치 여부 확인. 설치 경로가 다르면 `민원도우미_실행.bat` 파일 안의 `CHROME=` 경로를 수정. |
| 전체화면에서 나가고 싶음 | `Alt + F4` (관리자용). |
| 인쇄물 크기·위치가 이상함 | 프린터 설정에서 **여백 없음·배율 100%** 확인(제Ⅳ항). |
| 화면 카드가 안 보임 | 파일 8개가 **같은 폴더**에 함께 있는지 확인(index.html과 각 서식 파일). |
| 다른 사이트로 나가짐 | 방법 A(실행 파일=키오스크 모드)로 실행. |
