# 「군포시 민원 서식 작성 도우미」 사용 매뉴얼

〔붙임 2〕 · 작성일 2026. 7. 23. · 담당 〔부서명 / 담당자 / 연락처〕

## Ⅰ. 개요

- 본 도우미는 시민이 **쉬운 질문에 순서대로 답하면 실제 민원 서식이 자동으로 채워지는** 웹 기반 작성 안내 도구입니다.
- **설치가 필요 없으며**, 인터넷 브라우저(크롬 등)로 접속만 하면 됩니다.
- 입력 내용은 **저장·전송되지 않고** 화면을 닫으면 사라집니다. (개인정보 안전)

### 접속 방법
1. 인터넷 브라우저 주소창에 접속 주소 입력, 또는 **QR코드 스캔**.
   - 접속 주소 : **https://sura140705-crypto.github.io/gunpo-passport-helper/**
2. 첫 화면(메뉴)에서 작성할 서식을 선택합니다.

### 화면 구성
- **민원실 PC(넓은 화면)** : 왼쪽 = 실제 서식 미리보기 / 오른쪽 = 단계별 안내(코치).
- **스마트폰(좁은 화면)** : 안내(코치) 화면 중심, [🔍 완성된 서식 미리보기] 버튼으로 서식 확인.
- ※ [↔ 좌우 바꾸기] 버튼으로 서식·안내 위치를 바꿀 수 있습니다.

## Ⅱ. 공통 사용법 (모든 서식 동일)

### 1단계 — 서식 선택
- 첫 화면에서 원하는 서식 카드를 누릅니다. (준비 중 항목은 회색으로 표시되어 선택되지 않습니다.)

### 2단계 — 순서대로 답하기
- 오른쪽 안내창의 질문에 답하고 **[다음]** 을 누릅니다.
- 답하는 즉시 왼쪽 서식의 해당 칸이 채워지고, **지금 작성 중인 부분이 파란색으로 강조**됩니다.
- 어려운 용어는 **‘쉬운 말 풀이’** 를, 잘 모르는 항목은 **[🤔 잘 모르겠어요]** 를 눌러 건너뛴 뒤 창구에서 확인할 수 있습니다.
- **필수 항목**을 비우면 다음으로 넘어가지 않습니다(빨간 안내 표시).

### 3단계 — 미리보기
- **[미리보기]** 를 누르면 완성된 서식을 크게 확인할 수 있습니다.

### 4단계 — 인쇄 / PDF 저장 〔중요〕
- **[인쇄 / PDF 저장]** 을 누르면 인쇄 창이 열립니다.
- 인쇄 직전 안내 팝업이 뜨며, 반드시 아래 설정을 확인합니다.

> ### ⚠️ 인쇄 설정 (반드시 확인)
> - **용지 여백 : ‘없음’**
> - **배율(크기) : ‘실제 크기 100%’** (‘페이지에 맞춤’ 사용 금지)
>
> 이 설정을 지키지 않으면 인쇄물의 칸 위치·크기가 달라져 **여권 등 기계 판독에서 오류**가 날 수 있습니다.

### 기타 버튼
- **[초기화]** : 입력 내용을 모두 지우고 처음부터 다시 작성.
- **[작성예시]** : 예시 데이터를 자동으로 채워 작성 방법을 익힐 때 사용(실제 신청 시에는 [초기화] 후 본인 정보 입력).

## Ⅲ. 서식별 안내

### 1. 여권발급신청서 🛂
- **안내 항목** : 여권 종류, 사증 면수(26/58면)·유효기간, 신청인 정보, 로마자 성명(한글 이름 기반 예시 자동 제안), 긴급 연락처.
- **자동 처리** :
  - 주민등록번호 입력 시 **미성년자(만 18세 미만) 자동 판정** → 유효기간 5년 자동 적용·안내.
  - **미성년자** : 법정대리인 **동의서**가 함께 인쇄됨.
  - **대리 신청**(성인) : **위임장**이 함께 인쇄됨.
  - **예상 수수료·처리기간** 안내 표시.
- **주의** : 로마자 성명은 예시일 뿐 최종 확인은 본인 책임, 기존 여권이 있으면 영문명 일치 필요.

### 2. 혼인신고서 💍
- 혼인당사자(남편·아내) 정보, 부모 정보, **증인 2인**, 성·본 협의, 근친혼 여부 등.
- 주민등록번호 입력 시 출생연월일 자동 계산, 서명·날인란 형광 표시.
- 작성예시 : 성인 / 미성년(부모 동의자 자동 반영) 2종.

### 3. 출생신고서 👶
- 출생자(이름·성별·출생 일시·장소), 부모 정보, 신고인, 등록기준지.
- **혼인 중 / 혼인 외** 구분(혼인 외 출생 시 부 정보 생략 가능).
- 작성예시 : 혼인 중 / 혼인 외 2종.

### 4. 사망신고서 🕊️
- 사망자 정보, 사망 일시, **사망 장소 구분**(주택·의료기관·사회복지시설 등 10종), 신고인 자격·관계.
- 작성예시 : 의료기관 사망 / 자택 사망 2종.

### 5. 개명신고서 ✍️
- 개명 전·후 이름(한글·한자), 본, **개명허가일자·법원명**, 첨부(개명허가결정등본) 안내, 신고인 자격(본인·법정대리인·기타).
- 작성예시 : 본인 신고 / 법정대리인 신고 2종.

### 6. 이혼(친권자 지정)신고서 📄
- **협의상 / 재판상 이혼** 구분, 재판확정일자·법원명, **친권자 지정 자녀 최대 4명**(자녀별 친권자·효력발생일·원인).
- 작성예시 : 협의이혼 / 재판이혼 2종.

### 7. 증명서 발급신청(통합) 🧾 — 준비 중
- 가족관계·기본·혼인관계·입양·친양자 증명서를 한 곳에서 신청하도록 개발 예정.

## Ⅳ. 창구 직원용 안내

- **역할** : 본 도우미는 작성을 돕는 도구이며, **최종 접수·심사는 창구에서** 진행합니다.
- **고령층 지원** : 큰 글씨·쉬운 말 풀이가 기본 제공되나, 필요 시 [작성예시]를 함께 보며 설명하면 이해가 빠릅니다.
- **‘잘 모르겠어요’ 항목** : 마지막 요약 화면에 🤔 표시로 모여 표시되므로, 해당 항목만 창구에서 확인·보완하면 됩니다.
- **인쇄 오류 문의** : “인쇄물 크기·위치가 이상하다”는 문의는 대부분 **프린터 설정**(여백 없음·배율 100%) 문제이니 먼저 설정을 확인해 주세요.
- **개인정보** : 공용 PC·키오스크 이용 후 인쇄물·PDF가 남지 않도록 안내해 주세요(입력값은 화면을 닫으면 자동 소멸).

## Ⅴ. 자주 묻는 질문(FAQ)

| 질문 | 답변 |
|------|------|
| 입력한 정보가 저장되나요? | 아니요. 화면을 닫거나 새로고침하면 즉시 사라지며 서버에 전송·저장되지 않습니다. |
| 집에서 미리 작성할 수 있나요? | 네. 스마트폰·개인 PC에서 접속해 작성 후 인쇄·PDF 저장하여 방문하시면 됩니다. |
| 인쇄물이 실제 서식과 크기가 달라요. | 인쇄 설정에서 **여백 ‘없음’, 배율 ‘100%(실제 크기)’** 로 지정하세요. |
| 이 서식을 그대로 제출하면 접수되나요? | 작성 편의를 위한 도구이며, 최종 접수·수리는 창구 심사에 따릅니다. |
| 잘 모르는 항목이 있어요. | [🤔 잘 모르겠어요]로 건너뛰고, 창구에서 확인하시면 됩니다. |
