## 1. 추진 배경 및 필요성

### 가. 현황 및 문제점
- 민원실을 방문하는 시민이 여권발급신청서·가족관계등록 신고서 등 각종 서식을 **직접 손으로 작성**하는 과정에서 다음과 같은 어려움이 반복 발생함.
  - 서식 항목이 많고 용어가 어려워 **어느 칸에 무엇을 적어야 하는지** 파악이 어려움 (등록기준지, 로마자 성명, 사증 면수·유효기간, 친권자 지정 등).
  - **작성 오류·누락**으로 창구에서 반려·재작성이 잦아 민원 처리 시간이 지연됨.
  - 특히 **고령층 민원인**은 작은 글씨·복잡한 표 양식에 취약하여 직원의 1:1 안내 의존도가 높음.
  - 상황별로 필요한 **첨부 서류**(미성년자 여권의 법정대리인 동의서, 대리 신청 시 위임장 등)를 사전에 안내받기 어려움.

### 나. 개선 방향
- 시민이 **쉬운 질문에 순서대로 답하기만 하면** 실제 서식이 자동으로 채워지고, 곧바로 인쇄·PDF 저장할 수 있는 **웹 기반 작성 도우미**를 도입함.
- **개인정보를 일절 저장·전송하지 않는** 구조로 설계하여 정보보호 부담 없이 운영 가능함.
- 민원실 **키오스크·업무용 PC는 물론 시민 개인 스마트폰**에서도 접속 가능하게 하여, 방문 전 사전 작성까지 지원함.

### 다. 기대 필요성 요약
| 구분 | 도입 전 | 도입 후 |
|------|---------|---------|
| 작성 방법 안내 | 직원 구두 안내에 의존 | 단계별 자동 안내(코치) |
| 작성 오류·반려 | 빈번 | 필수항목 검증으로 감소 |
| 고령층 지원 | 직원 밀착 지원 필요 | 큰 글씨·쉬운 말 풀이 자체 제공 |
| 첨부서류 | 창구에서 뒤늦게 확인 | 상황별 자동 안내·자동 생성 |
| 개인정보 | — | 무저장·무전송(원천 차단) |

## 3. 서비스 구성 및 주요 기능

### 가. 제공 서식 (총 7종 서비스 중)

| 연번 | 서식명 | 주요 안내 내용 | 상태 |
|:---:|--------|----------------|:---:|
| 1 | 여권발급신청서 | 신규·재발급, 로마자 성명, 사증 면수·유효기간, **미성년자 동의서·대리 위임장 자동 생성**, 예상 수수료 안내 | 운영 |
| 2 | 혼인신고서 | 혼인당사자·증인 2인·부모 정보, 성·본 협의 | 운영 |
| 3 | 출생신고서 | 출생자·부모·신고인, 등록기준지, 혼인 중/외 구분 | 운영 |
| 4 | 사망신고서 | 사망자·신고인, 사망 일시·장소 구분 | 운영 |
| 5 | 개명신고서 | 개명 전·후 이름, 개명허가일자·법원, 신고인 자격 | 운영 |
| 6 | 이혼(친권자 지정)신고서 | 협의·재판상 이혼 구분, 친권자 지정 | 운영 |
| 7 | 증명서 발급신청(통합) | 가족관계·기본·혼인관계·입양·친양자 증명서 통합 신청 | **준비 중** |

### 나. 공통 핵심 기능
1. **단계별 작성 코치** — 화면 오른쪽에서 쉬운 질문을 순서대로 제시하고, 답하면 왼쪽 실제 서식에 실시간 반영.
2. **실제 서식 그대로 재현** — 공식 서식 이미지 위에 입력값을 정확한 위치에 배치하여, 인쇄 시 관공서 제출 양식과 동일한 형태로 출력.
3. **필수 항목 검증** — 빠뜨리면 안 되는 항목을 자동으로 확인하여 다음 단계 진행을 차단(모르는 항목은 '잘 모르겠어요'로 건너뛰고 창구 문의 가능).
4. **작성 예시 제공** — 상황별(성인/미성년자, 혼인 중/외 등) 예시를 한 번에 채워 보여주는 학습 기능.
5. **미리보기·인쇄·PDF 저장** — 완성된 서식을 확대 미리보기 후 인쇄하거나 PDF로 저장.
6. **상황별 첨부서식 자동 생성** — 미성년자 여권 시 법정대리인 동의서, 대리 신청 시 위임장을 자동으로 함께 인쇄.
7. **고령층 친화 설계** — 큰 글씨·고대비 색상·쉬운 말 풀이·명확한 진행 단계 표시.
8. **반응형 지원** — 민원실 PC(좌: 서식 / 우: 안내 2단 화면), 스마트폰(안내 중심 화면) 자동 전환.

### 다. 인쇄 정확도 안내 기능
- 기계 판독(여권 등)을 위해 **여백 없음·배율 100%(실제 크기)** 인쇄가 필요하며, 인쇄 직전 이를 안내하는 팝업을 제공하여 판독 오류를 예방함.

## 4. 개인정보 보호 방안 〔중점〕

> 본 서비스는 **개인정보 유출 위험을 구조적으로 차단**하도록 설계되어, 별도의 개인정보 처리 부담 없이 운영할 수 있음.

1. **무저장 원칙** — 시민이 입력한 이름·주민등록번호 등 모든 정보는 **접속한 브라우저 메모리에서만** 처리되며, 화면을 닫거나 새로고침하면 즉시 소멸함. 서버·데이터베이스·쿠키·로컬 저장소 등에 **일절 저장하지 않음**.
2. **무전송 원칙** — 입력 정보를 외부 서버로 **전송하지 않음**. 서식 작성·인쇄가 전부 이용자 기기 안에서 완결됨.
3. **암호화 접속** — 전 구간 HTTPS(암호화) 통신.
4. **결과물 관리** — 인쇄물·PDF는 이용자가 직접 관리하며, 키오스크·공용 PC 이용 후 파일이 남지 않도록 안내함.
5. **법적 성격** — 저장·수집이 없으므로 개인정보 수집·이용 동의 절차가 발생하지 않음(단, 운영 세부는 개인정보 담당부서 검토 권장).

## 5. 기대 효과

- **민원 처리 시간 단축** : 사전 작성·오류 감소로 창구 대기·재작성 최소화.
- **직원 업무 경감** : 반복적 작성 안내 부담 완화 → 상담·검토 업무에 집중.
- **시민 만족도 향상** : 고령층·디지털 취약계층도 스스로 서식 작성 가능.
- **비용 효율** : 별도 시스템 구축·서버 운영비 없이 운영(제7항 참조).
- **확장성** : 공통 엔진 기반이라 신규 서식 추가가 용이(설정 파일 1개로 서식 1종 추가).

## 6. 운영 방안

### 가. 접속 환경 구축
- **민원실 키오스크/PC** : 바탕화면 바로가기 또는 브라우저 시작페이지로 등록.
- **시민 스마트폰** : 창구·게시판에 **QR코드 포스터** 부착하여 즉시 접속 유도.
- **알기 쉬운 주소 정비**(권장) : 현재 주소가 여권 전용처럼 보이므로, 서식 플랫폼에 맞는 명칭으로 정비 검토(예: 저장소명 변경 또는 시 공식 도메인 연결).

### 나. 유지·관리
- 서식 개정·오류 수정 시 즉시 반영 가능(수정 즉시 자동 재배포).
- 서식 좌표·문구 등 세부 조정은 시범운영 중 현장 의견을 받아 순차 개선.

### 다. 안내·홍보
- 민원실 내 사용 안내문(붙임 2, 사용 매뉴얼) 비치.
- 창구 직원 대상 간단 사용법 공유.
